Sorting Algorithm Visualizer

This project is an educational tool, designed to help visualize the process of various sorting algorithms. Implemented in Python, the tool uses Pygame for the visual aspect and TKINTER for control. It generates arrays of squares and sorts them using Bubble, Insertion, Selection, Merge, and Quick sort algorithms. Explore the process of its creation, the logic behind the algorithms, and the code that drives it all.
